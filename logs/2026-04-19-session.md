## 3. Claude Code への MCP 登録

### 構成
`~/.claude.json` の `projects.<作業ディレクトリ>.mcpServers` に `meta-ads-local` を追加:
```json
{
  "type": "stdio",
  "command": "python",
  "args": ["-m", "meta_ads_mcp"],
  "env": {
    "META_APP_ID": "...",
    "META_ACCESS_TOKEN": "..."
  }
}
```

### 共存戦略
既存 Pipeboard MCP (`meta-ads`, `google-ads`) は削除せず並列稼働:
- ツール名プレフィックスで分離 (`mcp__meta-ads__*` vs `mcp__meta-ads-local__*`)
- Pipeboard週次リセット後は両方使える

### 再起動必要
Claude CodeはMCPを起動時のみロード。セッション終了→再起動で `mcp__meta-ads-local__*` 有効化。
