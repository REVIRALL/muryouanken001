# Evidence Vault（立証資料保管場所）

本ディレクトリは **景品表示法 7 条 2 項（不実証広告規制）** への対応のため、LP 上で表示している数値・実績・提携関係等の**合理的根拠を示す資料**を保管する場所です。

## 保管対象

LP 本文または広告クリエイティブで以下の種類の表示を行う場合、対応する立証資料を必ず本フォルダに保管してください。

- 「提携一次代理店 〇社」 → 提携契約書（写し）、事業者登記簿、Partner Program登録証
- 「累計紹介実績 〇件」 → 案件成約記録（個人情報マスク済）、請求ダッシュボード・スクショ
- 「代理店運営歴 〇年目」 → 事業開始証明（開業届、法人登記等）
- 「取引コミュニティ数 〇団体」 → 取引台帳、請求書ログ
- Voice 掲載（代理店の声）→ 本人同意書、実名確認書類
- 特商法記載の所在地 → バーチャルオフィス利用契約書

## 運用ルール

1. **消費者庁から 15 日以内に資料提出を求められた場合**、本フォルダから即時提出できる状態を常に維持する
2. ファイル命名: `<claim>_<date>.pdf` 形式推奨（例: `partner_A_contract_2026-04.pdf`）
3. **個人情報は必要最小限、マスク前提**
4. **plaintext の秘密情報（APIキー・トークン・パスワード等）は絶対に置かない**（gitignore対象）
5. 数値を LP に反映する前に、対応資料が本フォルダに存在することを `scripts/check_evidence.mjs` で検証する

## アクセス制限

本リポジトリは GitHub 上で **private** 管理を推奨（現状 public だが、資料追加前に private 化すること）。

## 保管期間

取引終了後 5 年（個情法・特商法の通例）。それ以降は適切に削除。
