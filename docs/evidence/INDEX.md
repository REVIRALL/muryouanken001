# Evidence Index（数値表記の合理的根拠一覧）

本ファイルは LP 本文に掲載している数値表記と、各数値を裏付ける立証資料との対応表です。
原本（請求書・契約書・登記簿・LINE ログ・台帳等）は **事業者にて社内ストレージで管理**しており、
本リポジトリには置いていません（個人情報・取引機密・取引先保護のため）。

景品表示法第 7 条 2 項の資料提出命令を受けた場合、**請求受付後 15 営業日以内**に書面で提出できるよう整備しています。

## 立証資料の社内保管場所

以下いずれも **非公開・社内ストレージ**で管理:

- `【リバイラル】請求書_株式会社イケメン_20250331.pdf`（2025年3月期、成約件数集計の根拠）
- `【リバイラル】支払明細書_*` 一式
- ASP 成果管理画面 スクリーンショット（2026-04-22 取得、2026年1-4月の月次成果数・売上額・クリック数を表示）
- ASP 複数社の月次実績 CSV / スクリーンショット（現在月平均の合算根拠）
- 法人登記事項証明書（2023年3月設立、2024年6月住所変更）
- LINE 業務用アカウントの取引ログ（竹本様／沼倉様／株式会社イケメン他）
- 取引先との契約書・NDA 原本

## 開示運用ルール

1. 消費者庁・都道府県消費生活センター等の公的機関からの資料提出命令 → **15 営業日以内**に書面で提出
2. 取引先・提携先に関する機密情報は、当該取引先の書面同意を得たうえで開示
3. 個人情報（氏名・連絡先等）は個人情報保護法に従いマスキング処理してから提出
