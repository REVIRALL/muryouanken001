■ MCP経由で投げる実コマンド例
================================================================

【Claude Code から nanobanana-mcp で編集リクエストする場合】

  mcp__nanobanana__nano_edit(
    image_path = "C:/Users/ooxmi/Downloads/C_shosentaku_v3_pro4k_master.jpg",
    prompt = "<A-2 の英語プロンプトをここに貼付>",
    output_path = "C:/Users/ooxmi/Downloads/revirall-meta-ads-automation/lp/assets/fv_v3_1080x1350.png",
    model = "gemini-3-pro-image-preview",
    aspect_ratio = "4:5"
  )

【Google AI Studio / Generative AI SDK で直接投げる場合（Python）】

  import google.generativeai as genai
  from pathlib import Path

  genai.configure(api_key="YOUR_GOOGLE_AI_API_KEY")
  model = genai.GenerativeModel("gemini-3-pro-image-preview")

  ref_image = genai.upload_file(
    path="C:/Users/ooxmi/Downloads/C_shosentaku_v3_pro4k_master.jpg",
    mime_type="image/jpeg",
  )

  prompt = """
  <A-2 の英語プロンプト全文>
  """

  response = model.generate_content([ref_image, prompt])
  for part in response.candidates[0].content.parts:
      if hasattr(part, "inline_data") and part.inline_data:
          Path("fv_v3_1080x1350.png").write_bytes(part.inline_data.data)

■ 生成後チェックリスト
================================================================

全画像共通:
  [ ] 元画像の温かみ・ベージュ色調が保たれている
  [ ] 被写体の顔・手が不自然に崩れていない
  [ ] 過剰な販促感（ネオン色・笑顔演出過多）が出ていない
  [ ] 透かし・ロゴ・ノイズ文字が混入していない

ルートA（テキスト無し）:
  [ ] 意図した位置に「余白」が確保されている
  [ ] 余白の色がHTML側のテキスト色と十分コントラストが取れる
  [ ] WebP/AVIF/JPEGで書き出して 200KB 以下

ルートB（テキスト有り）:
  [ ] 日本語が一文字も崩れていない
  [ ] 「売り込まない」「コミュニティ運営」「新しい選択肢」全部読める
  [ ] カーニング（字間）が不自然でない
  [ ] テキストが顔・手に被っていない
  [ ] Metaの画像内テキスト20%ルール抵触なし
      （文字占有面積をMeta Ads Managerで事前確認）

■ バッチ実行スクリプト（参考）
================================================================

【全サイズ一括生成の Python 例】

  import google.generativeai as genai
  from pathlib import Path

  genai.configure(api_key=YOUR_KEY)
  model = genai.GenerativeModel("gemini-3-pro-image-preview")
  ref = genai.upload_file(
    path="C:/Users/ooxmi/Downloads/C_shosentaku_v3_pro4k_master.jpg",
    mime_type="image/jpeg",
  )

  tasks = {
    "fv_1500x1200.png":   "<A-1 prompt>",
    "feed_1080x1350.png": "<A-2 prompt>",
    "sq_1080x1080.png":   "<A-3 prompt>",
    "story_1080x1920.png":"<A-4 prompt>",
  }

  out = Path("C:/Users/ooxmi/Downloads/revirall-meta-ads-automation/lp/assets/v3")
  out.mkdir(parents=True, exist_ok=True)

  for fname, prompt in tasks.items():
      resp = model.generate_content([ref, prompt])
      for part in resp.candidates[0].content.parts:
          if hasattr(part, "inline_data") and part.inline_data:
              (out / fname).write_bytes(part.inline_data.data)
              print(f"Saved: {fname}")
              break
